VS Code Remote SSH — Orange Pi Dev Setup

GCS Pi (192.168.50.1)  ·  Mast Pi (192.168.50.2)  ·  Windows PC (192.168.50.100)

Windows PC (192.168.50.100)
    │ USB-C Ethernet adapter
    │
  [Switch]
    │
  [Switch]
   ├── GCS Pi — 192.168.50.1  (enP4p65s0)
   └── Mast Pi — 192.168.50.2  (enP4p65s0)
1
Set static IP on the USB-C Ethernet adapter
  1. Press Win + R → type ncpa.cpl → Enter
  2. Find the USB-C Ethernet adapter (e.g. "Ethernet 2")
  3. Right-click → Properties
  4. Double-click Internet Protocol Version 4 (TCP/IPv4)
  5. Select Use the following IP address and enter:
FieldValueNote
IP address192.168.50.100Same subnet as Pis
Subnet mask255.255.255.0/24
Default gatewayLeave empty
Verify Open Command Prompt and run:
ping 192.168.50.1 and ping 192.168.50.2
Both should reply with 0% loss.
2
Generate SSH key pair on Windows
In Command Prompt: 
ssh-keygen -t ed25519 -N "" -f %USERPROFILE%\.ssh\id_ed25519
This creates a private key (id_ed25519) and public key (id_ed25519.pub) in C:\Users\<you>\.ssh\.
Verify dir %USERPROFILE%\.ssh
You should see both id_ed25519 and id_ed25519.pub.
3
Copy public key to both Pis
In Command Prompt, run both commands. Enter password orangepi when prompted. Accept the fingerprint prompt by typing yes. 
type %USERPROFILE%\.ssh\id_ed25519.pub | ssh orangepi@192.168.50.1 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"

type %USERPROFILE%\.ssh\id_ed25519.pub | ssh orangepi@192.168.50.2 "mkdir -p ~/.ssh && cat >> ~/.ssh/authorized_keys"
Verify — passwordless login ssh orangepi@192.168.50.1 and ssh orangepi@192.168.50.2
Both should connect without asking for a password. Type exit or press Ctrl+D to disconnect.
4
Configure SSH hosts
In Command Prompt: 
notepad %USERPROFILE%\.ssh\config
Add the following and save: 
Host gcs-pi
    HostName 192.168.50.1
    User orangepi

Host mast-pi
    HostName 192.168.50.2
    User orangepi
5
Connect with VS Code Remote SSH
  1. Open VS Code → Extensions (Ctrl+Shift+X) → search Remote - SSH → Install
  2. Press F1 → type Remote-SSH: Connect to Host
  3. Select gcs-pi — a new VS Code window opens connected to GCS Pi
  4. Repeat for mast-pi in a second VS Code window
  5. In each window: File → Open Folder → navigate to your project directory
Verify Bottom-left corner of each VS Code window shows SSH: gcs-pi and SSH: mast-pi in green.
